Local deportivo - Desarrollo y mantenimiento app web.

En este proyecto se me ha solicitado realizar una app web e-commerce para poder expandir las ventas de su negocio. Claramente se me han dado requerimientos como los colores a utilizar, alguna que otra interfaz que ya tenían pensada, por ejemplo el cómo se deberían de mostrar todos los productos, o cómo debería de verse el producto en detalle con su correspondiente descripción. Para esto se hizo uso de tecnologías como HTML, CSS, React y JavaScript.

Local deportivo - Desarrollo app de escritorio.

Recuerdo que este cliente me había solicitado un app de escritorio, tal como dice el título, para poder llevar un control contable digamos, con registros y gráficos sobre ventas realizadas en el día, total recaudado en el día, lo mismo para cada mes, y por producto, además de un control de stock de todos sus productos, con la posibilidad de poder actualizarlo por si ingresaran más productos. En este proyecto se utilizó Java para el desarrollo de la app, y se tuvo que realizar conexiones con la base de datos la cual ya tenían desarrollada, debido a que tienen también una web e-commerce.

Empresa de seguros - Desarrollo app de escritorio.

En este proyecto no trabajé solo, sino que con un conjunto de personas bastante numérico, ya que fuimos divididos para desarrollar y mantener una parte en específico de la app, se podría decir que éramos unos 5 por cada equipito asignado a un área. Biblioteca Popular - Desarrollo app de escritorio.

Este proyecto es algo parecido al de la tienda deportiva. Se me pidió realizar esa app de escritorio con Java, y consistía en una interfaz para llevar control de los libros que hay en la biblioteca, así como también una barra de búsqueda que permita tanto buscar por nombre como por ISBN, género, etc. También contaba con una parte para llevar un conteo o rastreo digamos, de los libros prestados, por ejemplo si estuviese cerca del día límite de prestado, se enviaría automáticamente un e-mail a modo de recordatorio.
